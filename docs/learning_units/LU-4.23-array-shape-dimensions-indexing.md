# Learning Unit 4.23: Understanding Array Shape, Dimensions, and Index Positions

## Objective

Understand how NumPy arrays are structured using shape and dimensions, and access elements using index positions.

## Product-Oriented Implementation

1. Create helper functions to inspect shape and dimensions for transaction arrays.
2. Demonstrate index access for key positions in 1D and 2D arrays.
3. Build notebook examples to avoid indexing errors in future anomaly features.

## Deliverables in this unit

- `src/fraudscope/numpy_shape_indexing.py`
- `notebooks/LU-4.23-array-shape-indexing.ipynb`
- LU implementation note and README update

## Why this matters for fraud detection

Correct indexing is required to extract account-level and feature-level values safely during anomaly scoring.
