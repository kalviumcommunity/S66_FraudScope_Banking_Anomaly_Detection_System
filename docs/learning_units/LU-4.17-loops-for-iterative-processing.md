# Learning Unit 4.17: Using for and while Loops for Iterative Data Processing

## Objective

Use loops to process batches of transaction records and repeated threshold checks.

## Product-Oriented Implementation

1. Use `for` loops for record-by-record processing and aggregation.
2. Use `while` loops for controlled repeated checks (e.g., scan until limit).
3. Demonstrate iterative methods in notebook examples tied to fraud scenarios.

## Deliverables in this unit

- `src/fraudscope/loop_processing.py`
- `notebooks/LU-4.17-loop-processing.ipynb`
- LU implementation note and README update

## Why this matters for fraud detection

Before vectorized tools, loops help create clear baseline logic for velocity checks, rolling scans, and event counting.
