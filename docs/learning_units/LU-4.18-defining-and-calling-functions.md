# Learning Unit 4.18: Defining and Calling Python Functions

## Objective

Create reusable Python functions for transaction scoring and call them from analysis workflows.

## Product-Oriented Implementation

1. Define small focused functions for risk scoring and reason generation.
2. Call functions in sequence to create consistent output records.
3. Demonstrate function usage in a notebook with sample transaction data.
